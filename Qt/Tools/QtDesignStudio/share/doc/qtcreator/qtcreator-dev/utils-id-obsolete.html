Obsolete Members for Id

[static] Utils::Id Id::fromName(const QByteArray &name)

This function is deprecated. We strongly advise against using it in new code.

Creates an id from a string representation.

This should not be used to handle a persistent version of the Id, use fromSetting() instead.

See also toString() and fromSetting().
