[static] bool FileUtils::copyIfDifferent(const Utils::FilePath &srcFilePath, const Utils::FilePath &tgtFilePath)

Copies a file specified by srcFilePath to tgtFilePath only if srcFilePath is different (file contents and last modification time).

Returns whether the operation succeeded.
