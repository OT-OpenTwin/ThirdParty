Debugging and Profiling

  • Debugging Qt Quick Projects

    You can use the integrated debugger to debug JavaScript functions, to execute JavaScript expressions to get information about the state of the UI, and to inspect QML properties and JavaScript variables and change them temporarily at runtime.

  • Debugging a Qt Quick Example Application

    This section uses the Same Game example application to illustrate how to debug Qt Quick applications in the Debug mode.

  • Profiling QML Applications

    You can use the QML Profiler to find causes for typical performance problems in your UIs, such as slowness, freezing, and stuttering. The QML Profiler is installed and configured as part of Qt Design Studio.
