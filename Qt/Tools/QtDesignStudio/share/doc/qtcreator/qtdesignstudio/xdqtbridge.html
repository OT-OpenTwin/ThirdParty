Exporting Designs from Adobe XD

Note: This is a Technical Preview release of the Qt Bridge for Adobe XD. Some design elements might not be exported and imported into Qt Design Studio as expected.

You can use Qt Bridge for Adobe XD to export designs from Adobe XD to .qtbridge format that you can import to projects in Qt Design Studio.

The following topics describe setting up and using Qt Bridge for Adobe XD:

  • Setting Up Qt Bridge for Adobe XD

    You must install and set up the Qt Bridge for Adobe XD export plugin before you can use it to export designs.

  • Using Qt Bridge for Adobe XD

    To get the best results when you use Qt Bridge to export designs from Adobe XD, you should follow the guidelines for working with Adobe XD and organizing your assets.
