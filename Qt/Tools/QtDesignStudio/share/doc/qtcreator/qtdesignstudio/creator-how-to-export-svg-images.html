Export SVG images

If you have a freely scalable icon in the SVG format, you can open it in the image viewer to export it to several images of different sizes to create a set of pixmaps.

Select on the toolbar.

You can then use QIcon::addPixmap() to add the pixmaps to icons in different modes and states.
