Importing Designs

You can import 2D and 3D assets to Qt Design Studio projects. You need to use Qt Bridge for exporting the 2D assets from design tools, whereas you can directly import 3D assets saved in widely-used 3D graphics formats.

  • Importing 2D Assets

    You can import 2D assets that you exported with Qt Bridge to a Qt Design Studio project as image and QML files and edit them in the Design mode.

  • Importing 3D Assets

    You can import 3D assets that you created using 3D graphics applications and stored in one of the supported file formats. You can then edit them in the Design mode.
