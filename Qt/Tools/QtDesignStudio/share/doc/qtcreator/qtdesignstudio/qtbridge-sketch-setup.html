Setting Up Qt Bridge for Sketch

Qt Bridge for Sketch is included in the Qt Design Studio Enterprise license. You can use Qt Online Installer to have the Qt Bridge for Sketch plugin package copied to the following path in your Qt installation folder:

  • On Windows: Tools\QtDesignStudio\sketch_bridge\io.qt.qtbridge.sketchplugin
  • On macOS: QtDesignStudio/sketch_bridge/io.qt.qtbridge.sketchplugin.

Note: Install the Sketch app before installing the plugin.

To install the Qt Bridge for Sketch plugin to Sketch, double-click io.qt.qtbridge.sketchplugin in the sketch_bridge folder in the installation directory of Qt Design Studio. Sketch will automatically install Qt Bridge for Sketch.

You can launch the Sketch plugin from Plugins > Qt Bridge in Sketch.
