Qt Design Studio on MCUs

With Qt Design Studio, you can use subsets of components to create UIs for devices that are powered by microcontroller units (MCU). Use the Qt for MCUs preset in the Qt Design Studio wizard to set up a new Qt for MCUs project. Using the Qt for MCUs preset creates an application that utilizes a subset of the default components that you can deploy, run, and debug on MCU boards.
