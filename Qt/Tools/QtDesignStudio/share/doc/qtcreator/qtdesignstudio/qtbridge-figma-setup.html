Setting Up Qt Bridge for Figma

Note: Qt Bridge for Figma is included in the Qt Design Studio Enterprise license.

You need both Figma and Qt accounts to use Qt Bridge for Figma in Qt Design Studio. To use Qt Bridge for Figma in Qt Design Studio:

  1. Go to Qt Bridge for Figma plugin page.
  2. Select Try it out.
  3. Select your logged in Figma account.
  4. After the plugin loads, select Run.
  5. Select Export, to get the .qtbridge file in your local drive.
  6. In Qt Design Studio, drag the file to the 2D, 3D, Assets, or Navigator view in an open project .

You can launch the installed Figma plugin from Plugins > Qt Bridge for Figma in Figma.
