Import and export keyboard shortcuts

You can use different keyboard shortcut mapping schemes that are stored as .kms files.

To import and export keyboard shortcut mapping schemes:

  1. Select Preferences > Environment > Keyboard.

  2. To import a keyboard shortcut mapping scheme, click Import and select the .kms file that has the keyboard shortcut mapping scheme you want to import.
  3. To export the current keyboard shortcut mapping scheme, click Export and select the location where you want to save the exported .kms file.

See also Keyboard Shortcuts, Assign keyboard shortcuts, and Find keyboard shortcuts.
