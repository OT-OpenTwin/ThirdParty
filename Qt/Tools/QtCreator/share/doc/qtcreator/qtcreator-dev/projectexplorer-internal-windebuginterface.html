Header: #include <WinDebugInterface>

Detailed Description

The WinDebugInterface class is used on Windows to capture output of the Windows API OutputDebugString() function.

Emits output by process id.

The OutputDebugString() function puts its data into a shared memory segment named DBWIN_BUFFER which can be accessed via file mapping.
