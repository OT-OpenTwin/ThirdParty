Edit like in Vim

To run the main editor in a manner similar to the Vim editor, go to Edit > Advanced and select Use FakeVim, or press Alt+Y,Alt+Y.

Escape FakeVim mode

To temporarily escape FakeVim mode to access the normal Qt Creator keyboard shortcuts like Ctrl-R for Run, press the comma key (,) before pressing the shortcut.

Quit FakeVim mode

To quit the FakeVim mode, go to Edit > Advanced and clear Use FakeVim, or press Alt+Y,Alt+Y.

See also FakeVim Modes and Commands, FakeVim, and Change editor colors.
