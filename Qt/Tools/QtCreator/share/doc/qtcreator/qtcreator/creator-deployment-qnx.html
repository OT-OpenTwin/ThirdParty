Deploying to QNX Neutrino

You can specify settings for deploying applications to QNX Neutrino devices in the project configuration file and in Projects > Run Settings > Deployment.

The deployment process is described in more detail in Deploying to Remote Linux.

Finding Configured Devices

The Check for a configured device deployment step looks for a device that is ready for deployment.
