Add a Java language server

To add a Java language server for developing Android applications:

  1. Go to Preferences > Language Client > Add > Java Language Server to add a Java language server.

  2. In the Name field, enter a name for the language server. Select the (Variables) button to use a variable for the server name.
  3. In the Java field, enter the path to the Java executable.
  4. In the Java Language Server field, enter the path to the Java language server .jar file. You can download the Java language server from Eclipse Downloads.

See also How To: Manage Language Servers, Use Qt Creator variables, and Language Servers.
