Create Google tests

To create a Google test:

  1. Go to File > New Project > Test Project.
  2. Select Google Test Project > Choose to create a project with boilerplate code for a Google test.
  3. In the Project and Test Information dialog, specify settings for the project and test:

    1. In Test framework, select Google Test (shared libraries) to link against Google Test or Google Test (headers only) to include necessary Google Test sources into the project.
    2. In Test suite name, enter a name for the test suite.
    3. In Test case name, enter a name for the test case.
    4. For a shared library test, you can set the path to a Google C++ testing framework installation in Googletest install directory (optional).
    5. For a header-only test, you can set the path to a googletest repository clone in Googletest source directory (optional).
    6. In Build system, select the build system to use for building the project: CMake, qmake, or Qbs.

Qt Creator creates the test in the specified project directory.

For more information about creating Google tests, see the Google Test Primer.

See also How To: Test, Select the build system, Testing, and Test Results.
